void DataLoggerSD::logData ( bool  includeSeconds)

Logs data in a file, the files are designed to be viewed on a system chronologically.

The file name is the current year + the abbreviated month over which the data was collected. This logs all the data.

Parameters
includeSecondsis if the log is to include seconds.
