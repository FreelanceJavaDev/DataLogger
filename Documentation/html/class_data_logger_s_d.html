DataLoggerSD::DataLoggerSD ( uint8_t  sdPin)

The simple version of the DataLogger constructor.

Takes sensor readings every 5 minutes. Interrupt Pin is set to digital pin 2.

Parameters
sdPinis the pin for the SD card defined on the shield.

The custom DataLogger constructor.

For maximum compatiability select either digital pin 2 or 3 for the interrupt pin.

Parameters

minutesdefines how many minutes between sensor readings, minimum is one minute.

void DataLoggerSD::logData ( bool  includeSeconds)

Logs data in a CSV file (Excell compatible), the files are designed to be viewed on a system chronologically.

The file name is the current year + the abbreviated month over which the data was collected.

Parameters
includeSecondsis if the log is to include seconds.
